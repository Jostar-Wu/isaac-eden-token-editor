# 安全与问题报告

## 隐私

程序完全离线，不上传存档、不使用遥测，也不需要账户。

## 报告问题

请通过 GitHub Issue 报告问题，但不要公开上传真实存档。建议提供：

- 程序版本；
- Repentance 或 Repentance+；
- 使用自动模式还是手动模式；
- 界面显示的中文错误信息；
- 已删除用户名、Steam ID 和完整路径的截图。

如果怀疑存档损坏，请立即停止重复写入并恢复 `文档\IsaacEdenEditorBackups` 中最近的备份。

## 支持范围

安全更新只覆盖最新 GitHub Release。项目不保证兼容游戏未来新增的存档格式；未知版本会被拒绝，而不会尝试猜测性修改。
